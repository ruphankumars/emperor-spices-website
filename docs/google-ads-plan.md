## 5. WhatsApp conversion (the goal)

Google Search has no native "WhatsApp" button, so use BOTH:
1. **Final URL → your /contact page**, which already has a WhatsApp button (+91 97900 05649). Add a
   query param like `?utm_source=googleads&utm_campaign=me_bulk` so you can see ad-driven enquiries.
2. **Call asset (extension)** with +91 97900 05649 for direct calls.
3. **Sitelink assets:** /export/bulk-cardamom-suppliers, /products, /export/cardamom-grades-ageb-agb-ags, /export/cardamom-export-middle-east
4. **Conversion tracking:** in Google Ads → Goals → create a conversion for clicks on the WhatsApp
   link. Easiest: add the Google Ads tag (or use GA4 linked to Ads) and track the wa.me click as an
   event. (This needs a small snippet on the site — I can add the GA4/Ads tag + WhatsApp-click event
   when you have the conversion ID.)

Tip: a **wa.me deep link** as the Final URL works on mobile (`https://wa.me/919790005649?text=Hi%2C%20I%27m%20an%20importer%20interested%20in%20bulk%20green%20cardamom`), but desktop users need the landing page — so the landing-page-with-WhatsApp-button route is safer for B2B.
